# Google Keep Clone (React + Vite)

A Google Keep-style note-taking app built with React and Vite.

## Features

- Create notes with a title and content
- Archive/unarchive notes
- Pin/unpin notes to keep them at the top
- Copy note content to clipboard
- Edit existing notes in a modal
- Search notes instantly
- Light/Dark mode toggle with persistence
- Google Keep-inspired light theme and modern dark mode
- `Kokesto dark mode feature` added for theme switching and local storage persistence

## Installation

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Then open the local dev URL in your browser.

## Theme toggle

Use the `Light mode` / `Dark mode` button in the header to switch themes. The selected mode is saved in local storage so it remains active on refresh.

## Notes

This project uses the local browser storage to keep notes between sessions. No backend is required.
